/********** Features of app **********/
1. Create,Read,Update,Delete(CRUD) Tasks
2. Used React Hook from with validation
3. Used Redux RTK Query for managing state & http request.
4. Use framer motion for smooth transition.
5. User reusable component.
6. Home page,TaskLists page,TaskAdd,TaskEdit etc
7. Authentication using Jwt token & cookies.
8. Login,signup.
9. Protected routes.
10.Responsive ui across all devices.
11. Use tailwindcss for eligant styling.
12. Implement filter taks  based on Status and  Priority
    Note :  To see result click on status then priority 
            you will get desired result.



/##### Steps to run Client(Frontend)
Step 1. Go to folder Frontend.
Step 2. npm install
Step 3. npm run dev


/##### Steps to run Backend(Server)
Step 1. Go to folder server.
Step 2. make .env file and add your MONGO_URL there.
Step 3. npm install
Step 4. npm run dev
